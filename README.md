# Equipment Tracker

A simple full-stack Equipment Tracker application built using React and Node.js.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: JSON file
- HTTP Client: Axios

## Features

- View equipment list
- Add equipment
- Edit equipment
- Delete equipment

## How to Run Locally

### Backend

```bash
cd backend
npm install
node server.js
